### ${word}()
Liefert die nächste Linie aus einer geöffneten Datei. Leerstring, wenn keine solche vorhanden ist.

----

### Arguments
```ts
channelNo : number
```
#### Usage:
```ts
var stringVal := READ.NextLine(channelNo)
```
