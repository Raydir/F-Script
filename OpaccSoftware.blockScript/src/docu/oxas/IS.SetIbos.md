### ${word}()
IBOS-Name für den IBOS-Aufruf setzen. Es kann definiert werden, ob vormals vorhandene Argumente erhalten bleiben sollen.

----

### Arguments
```ts
ibosName : string
clearArgs : boolean default: 0
```
#### Usage:
```ts
IS.SetIbos(ibosName, clearArgs)
```
