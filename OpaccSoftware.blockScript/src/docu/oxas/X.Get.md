### ${word}()
Wert einer bestimmten X-Variable abholen.

----

### Arguments
```ts
xVarName : string
errorIfUndefined : boolean default: 0
```
#### Usage:
```ts
var stringVal := X.Get(xVarName, errorIfUndefined)
```
