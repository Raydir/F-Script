### ${word}()
Initialisiert Zeilenumbrüche und den Einzug.

----

### Arguments
```ts
lineBreakActive : boolean
indentActive : boolean
indentWidth : number default: 0
```
#### Usage:
```ts
JSON.Initialize(lineBreakActive, indentActive, indentWidth)
```
