### ${word}()
Filter zurücksetzen. Zurückgegeben wird die Anzahl Zeilen welche nach dem Entfernen des Filters sichtbar ist (alle Zeilen).

----

### Arguments
```ts
xTabName : string
```
#### Usage:
```ts
var numberVal := XTAB.ClearViewFilter(xTabName)
```
