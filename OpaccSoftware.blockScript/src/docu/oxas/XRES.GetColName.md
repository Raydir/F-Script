### ${word}()
Liefert Spalten-Namen anhand Spalten-Nummer. Fehler wenn nicht vorhanden

----

### Arguments
```ts
xResName : string
oneBasedColNo : number default: 1
```
#### Usage:
```ts
var stringVal := XRES.GetColName(xResName, oneBasedColNo)
```
