### ${word}()
Wert zum letzten Service-Argument hinzufügen. Fehler, wenn noch kein Argument vorhanden ist.

----

### Arguments
```ts
xSerName : string
argValueToAppend : string
```
#### Usage:
```ts
XSER.AppendArg(xSerName, argValueToAppend)
```
