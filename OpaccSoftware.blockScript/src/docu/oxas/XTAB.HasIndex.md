### ${word}()
Prüfen ob Index mit der übergebenen Nummer vorhanden ist.

----

### Arguments
```ts
xTabName : string
oneBasedIndexNo : number
```
#### Usage:
```ts
var boolVal := XTAB.HasIndex(xTabName, oneBasedIndexNo)
```
