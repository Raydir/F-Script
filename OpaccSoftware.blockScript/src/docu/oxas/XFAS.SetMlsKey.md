### ${word}()
MLS-Meldungs-Schlüssel setzen (setzt allfällig gesetzte Parameter zurück). Abkürzung für 'XFAS(mMlsKey=key)'.

----

### Arguments
```ts
mlsKey : string
```
#### Usage:
```ts
XFAS.SetMlsKey(mlsKey)
```
