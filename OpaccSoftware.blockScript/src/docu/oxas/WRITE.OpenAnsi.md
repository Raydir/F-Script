### ${word}()
Ausgabekanal zum Schreiben einer ANSI-Datei öffnen. Standardmässig wird allenfalls bestehende Datei überschrieben.

----

### Arguments
```ts
channelNo : number
filePath : string
append : boolean default: 0
```
#### Usage:
```ts
WRITE.OpenAnsi(channelNo, filePath, append)
```
