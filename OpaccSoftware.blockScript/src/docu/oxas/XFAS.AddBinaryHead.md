### ${word}()
Eine Spalte für die Rückgabe von Binärdaten hinzufügen. Als Werte müssen Dateinamen mit Pfad abgefüllt werden. Abkürzung für 'XFAS(headBinary=name)'.

----

### Arguments
```ts
head : string
```
#### Usage:
```ts
XFAS.AddBinaryHead(head)
```
