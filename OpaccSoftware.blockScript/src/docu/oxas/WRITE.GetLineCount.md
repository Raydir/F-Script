### ${word}()
Liefert Anzahl Linien welche in den Kanal geschrieben wurden = Anzahl 'WRITE.OutLine()'-Aufrufe.

----

### Arguments
```ts
channelNo : number
```
#### Usage:
```ts
var numberVal := WRITE.GetLineCount(channelNo)
```
