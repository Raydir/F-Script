### ${word}()
Wert einer bestimmten X-Variable ergänzen.

----

### Arguments
```ts
xVarName : string
xVarValueToAppend : string
errorIfUndefined : boolean default: 0
```
#### Usage:
```ts
X.Append(xVarName, xVarValueToAppend, errorIfUndefined)
```
