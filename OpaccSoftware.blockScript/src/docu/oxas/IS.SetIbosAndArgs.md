### ${word}()
IBOS-Name für den IBOS-Aufruf und IBOS-Argumente setzen. Allfällig vormals vorhandene Argumente werden zurückgesetzt.

----

### Arguments
```ts
ibosName : string
ibosArgs : string
```
#### Usage:
```ts
IS.SetIbosAndArgs(ibosName, ibosArgs)
```
