### ${word}()
Konvertiert mehrere Elemente in eine gültige CSV-Eintrags-Linie.

----

### Arguments
```ts
delimiter : string
elementObjects : string
```
#### Usage:
```ts
var stringVal := CSV.CreateLine(delimiter, elementObjects)
```
