### ${word}()
Wert für eine bestimmte Spalte setzen. Abkürzung für 'XFAS(col=ColName,value)'.

----

### Arguments
```ts
colName : string
value : string
```
#### Usage:
```ts
XFAS.SetCol(colName, value)
```
