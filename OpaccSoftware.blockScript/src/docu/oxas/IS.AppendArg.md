### ${word}()
Wert zum letzten IBOS-Argument hinzufügen. Fehler, wenn noch kein Argument vorhanden ist.

----

### Arguments
```ts
argValueToAppend : string
```
#### Usage:
```ts
IS.AppendArg(argValueToAppend)
```
