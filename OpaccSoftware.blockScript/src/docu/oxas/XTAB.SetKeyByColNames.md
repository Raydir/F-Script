### ${word}()
Schlüssel gemäss den übergebenen Spalten setzen. Definiert automatisch einen 'Unique - Constraint'. ACHTUNG: Kann bei Tabellen mit vielen Zeilen langsam sein!

----

### Arguments
```ts
xTabName : string
colNames : string
```
#### Usage:
```ts
XTAB.SetKeyByColNames(xTabName, colNames)
```
