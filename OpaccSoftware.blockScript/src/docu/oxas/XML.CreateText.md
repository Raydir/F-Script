### ${word}()
Konvertiert einen Text in ein gültiges XML-Format.

----

### Arguments
```ts
xmlText : string
suppressIndent : boolean default: 0
```
#### Usage:
```ts
var stringVal := XML.CreateText(xmlText, suppressIndent)
```
