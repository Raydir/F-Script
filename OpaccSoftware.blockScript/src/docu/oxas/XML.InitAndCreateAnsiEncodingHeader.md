### ${word}()
Retourniert den ANSI XML-Header und initialisiert den Einzug.

----

### Arguments
```ts
activateIndent : boolean
numberOfSpaces : number default: 0
```
#### Usage:
```ts
var stringVal := XML.InitAndCreateAnsiEncodingHeader(activateIndent, numberOfSpaces)
```
