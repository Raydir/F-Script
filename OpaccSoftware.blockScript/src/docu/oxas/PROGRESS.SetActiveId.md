### ${word}()
Setzt ein laufendes Monitor-Objekt für die Ausführung. Fehler, falls bereits ein laufendes Monitor-Objekt existiert.

----

### Arguments
```ts
progressId : number
```
#### Usage:
```ts
PROGRESS.SetActiveId(progressId)
```
