### ${word}()
Service-Argumente setzen. Allfällig vormals vorhandene Argumente werden zurückgesetzt.

----

### Arguments
```ts
xSerName : string
serviceArgs : string
```
#### Usage:
```ts
XSER.SetArgs(xSerName, serviceArgs)
```
