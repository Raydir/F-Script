### ${word}()
XSer-Objekt für einen bestimmten Service-Aufruf erzeugen (falls nötig). Allfällig vormals vorhandene Argumente bleiben erhalten.

----

### Arguments
```ts
xSerName : string
servicePortAndName : string
```
#### Usage:
```ts
XSER.SetService(xSerName, servicePortAndName)
```
