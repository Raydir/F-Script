### ${word}()
Erzeugt einen neuen Schritt. Optional kann verhindert werden, dass der Abbruch verlangt werden kann.

----

### Arguments
```ts
name : string
cancelSupported : boolean default: 0
```
#### Usage:
```ts
PROGRESS.CreateStep(name, cancelSupported)
```
