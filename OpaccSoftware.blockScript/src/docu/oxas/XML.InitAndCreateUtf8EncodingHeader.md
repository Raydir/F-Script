### ${word}()
Retourniert den UTF-8 XML-Header und initialisiert den Einzug.

----

### Arguments
```ts
activateIndent : boolean
numberOfSpaces : number default: 0
```
#### Usage:
```ts
var stringVal := XML.InitAndCreateUtf8EncodingHeader(activateIndent, numberOfSpaces)
```
