### ${word}()
Liefert Anzahl Zeilen (standardmässig unter Einbezug einer allfälligen View).

----

### Arguments
```ts
xTabName : string
withoutView : boolean default: 0
```
#### Usage:
```ts
var numberVal := XTAB.GetNmbOfRows(xTabName, withoutView)
```
