### ${word}()
Bestimmten Eintrag entfernen. Fehler, wenn Eintrag nicht vorhanden ist. Geliefert wird neue Anzahl Einträge.

----

### Arguments
```ts
xArrName : string
oneBasedItemIndex : number
```
#### Usage:
```ts
var numberVal := XARR.DeleteItem(xArrName, oneBasedItemIndex)
```
