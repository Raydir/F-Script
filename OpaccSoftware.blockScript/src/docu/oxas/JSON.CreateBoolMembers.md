### ${word}()
Erstellt mehrere JSON-Members mit je einem bool'schen Eintrag.

----

### Arguments
```ts
nameAndValues : any
```
#### Usage:
```ts
var stringVal := JSON.CreateBoolMembers(nameAndValues)
```
