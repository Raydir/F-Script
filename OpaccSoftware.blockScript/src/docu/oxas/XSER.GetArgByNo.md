### ${word}()
Liefert den Wert eines bestimmten Arguments. Fehler, wenn Argument nicht vorhanden ist.

----

### Arguments
```ts
xSerName : string
oneBasedArgNo : number
```
#### Usage:
```ts
var stringVal := XSER.GetArgByNo(xSerName, oneBasedArgNo)
```
