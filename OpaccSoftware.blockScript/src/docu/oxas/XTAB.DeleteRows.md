### ${word}()
Alle Zeilen löschen (standardmässig unter Einbezug einer allfälligen View). Zurückgegeben wird die Anzahl gelöschter Zeilen.

----

### Arguments
```ts
xTabName : string
withoutView : boolean default: 0
```
#### Usage:
```ts
var numberVal := XTAB.DeleteRows(xTabName, withoutView)
```
