### ${word}()
Sortierreihenfolge gemäss den übergebenen Spalten-Namen setzen. Prefix '-' = absteigende Sortierung. ACHTUNG: Kann bei Tabellen mit vielen Zeilen langsam sein!

----

### Arguments
```ts
xTabName : string
colNames : string
```
#### Usage:
```ts
XTAB.SetViewOrderByColNames(xTabName, colNames)
```
