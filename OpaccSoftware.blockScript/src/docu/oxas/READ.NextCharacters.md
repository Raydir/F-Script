### ${word}()
Liefert die nächsten Zeichen aus einer geöffneten Datei. Leerstring, wenn keine solchen mehr vorhanden sind.

----

### Arguments
```ts
channelNo : number
nmbOfCharacters : number
```
#### Usage:
```ts
var stringVal := READ.NextCharacters(channelNo, nmbOfCharacters)
```
