### ${word}()
Erstellt mehrere JSON-Members mit je einem Text-Eintrag.

----

### Arguments
```ts
nameAndValues : string
```
#### Usage:
```ts
var stringVal := JSON.CreateStrMembers(nameAndValues)
```
